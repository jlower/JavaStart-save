* 比如：Math类：sqrt()\random() \...

*     Scanner类：nextXxx() ...

*                3.2 返回值类型： 返回值  vs 没返回值

*                        3.2.1  如果方法返回值，则必须在方法声明时，指定返回值的类型。同时，方法中，需要使用

*                return关键字来返回指定类型的变量或常量：“return 数据”。

*                                  如果方法没返回值，则方法声明时，使用void来表示。通常，没返回值的方法中，就不需要

*               使用return.但是，如果使用的话，只能“return;”表示结束此方法的意思。
